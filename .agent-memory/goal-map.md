## Target Definition
**Goal:** Построить Marketing Factory — полностью автоматизированную систему от ввода продукта/рынка до набора PDF-отчётов (1-2 стр каждый) по каждому этапу: рынок, VOC (мин 300 отзывов), конкуренты, копирайт, креативы. Качество минимум 8.5/10 по стандартам топ-агентства.
**Success Criteria:**
- [ ] Каждый этап автоматически выдаёт PDF 1-2 страницы с ключевыми данными
- [ ] VOC этап собирает минимум 300 отзывов/сообщений рынка
- [ ] Система запускается одним вводом (продукт + рынок) без ручных шагов между этапами
- [ ] Качество каждого вывода оценивается честно — минимум 8.5/10
- [ ] Промпты для Perplexity покрывают и meta-уровень (как строить) и операционный (конкретный рынок)
- [ ] Все этапы чейнятся: вывод этапа N становится входом этапа N+1
**Failure Criteria (когда останавливаемся):**
- Любой этап требует ручного вмешательства пользователя
- PDF генерируется с общими данными, не специфичными для продукта
- VOC менее 300 источников
- Оценка качества ниже 8.5/10
**Current Actual Behavior:** Два standalone агента (copywriting_agent_tool.py, creative_agent_tool.py). Нет автоматического сбора данных. Нет PDF вывода. Нет пайплайна между этапами. Нет Perplexity интеграции. Нет operational промптов.
**Gap:** Отсутствует: автоматический сбор данных (Apify/Perplexity API), PDF рендеринг, оркестрация пайплайна, операционные промпты под конкретный продукт, human review gates, chaining этапов
